KPI Scoring Criteria

  • 1. Fear & Greed Index

    Score = raw value (0–100).

  • 2. CBBI Index

    Score = raw value (0–100).

  • 3. BTC 200‑Week MA

    Compares current BTC price to the 200‑week moving average.
    0 → BTC price = 85% of 200W MA (15% below).
    100 → BTC price = 220% of 200W MA (120% above).
    Linear scaling in between.

  • 4. 2‑Year MA Multiplier

    Compares current BTC price to the 2‑year MA and the 2‑year MA ×5.
    0 → BTC price < 66% of 2Y MA.
    100 → BTC price > 150% of (2Y MA ×5).
    Linear scaling in between.

  • 5. Pi Cycle Bottom

    Compares 150‑day EMA to 471‑day SMA (raw, before 0.745 multiplier).
    0 → EMA150 ≤ SMA471 (ratio ≤ 1).
    100 → EMA150 ≥ 2 × SMA471 (ratio ≥ 2).
    Linear scaling in between.

  • 6. Coinbase App Rank (Finance Apps)

    0 → Finance rank ≥ 50 (worst).
    100 → Finance rank ≤ 1 (best).
    Linear scaling: (50 – rank) / 49 × 100.

  • 7. BTC Market Dominance

    0 → Dominance ≥ 66% (most dominance).
    100 → Dominance ≤ 40% (least dominance).
    Linear scaling: (66 – dominance) / (66 – 40) × 100.

  • 8. Bitcoin Rainbow Chart

    Maps the 9 bands to a 0–100 scale:
    Basically a Fire Sale → 0
    BUY! → ~14
    Accumulate → ~28
    Still cheap → ~42
    HODL! → ~56
    Is this a bubble? → ~70
    FOMO intensifies → ~84
    Sell. Seriously, SELL! → 100
    Maximum Bubble Territory → 100
